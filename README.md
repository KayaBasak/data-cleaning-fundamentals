# Data Cleaning Fundamentals

A comprehensive collection of articles and hands-on examples covering the essential concepts of data cleaning with **Python** and **Pandas**.

This repository accompanies the **Data Cleaning Fundamentals** blog series, where each article focuses on a key aspect of preparing raw data for analysis. In addition to the articles, the repository includes a complete end-to-end data cleaning notebook that demonstrates how these techniques can be applied to a real-world dataset.

**Topics covered**

* Types of data quality problems
* Common causes of dirty data

## 💻 End-to-End Data Cleaning Notebook

This repository also includes a complete Google Colab notebook demonstrating an end-to-end data cleaning workflow using a real-world dataset.

### Dataset

**Global Layoffs Dataset (2020–2023)**

The notebook walks through the entire cleaning process, including:

* Loading the raw dataset
* Inspecting the data
* Handling missing values
* Removing duplicate records
* Fixing data types
* Investigating outliers
* Validating the cleaned dataset
* Exporting the cleaned dataset

The notebook serves as a practical companion to the blog series, showing how the individual techniques can be combined into a reproducible data cleaning pipeline.
